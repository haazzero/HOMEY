## ⚙️ 개발 환경
- `Java 11`
- `java-11-openjdk-11.0.15-1`
- **OS** : Window 10
- **IDE** : eclipse(sts-3.9.17)
- **Framework** : Spring(5.2.24)
- **Database** : Oracle DB(19.3.0.0.0)(11g)
- **ORM** : Mybatis
- **Application server** :  Apache Tomcat® v9.0
- **형상관리** : Github
